# Public Portfolio Checklist

This repository is a public portfolio case study. The private source repository should remain private unless a separate full release review is approved.

## Included

- Portfolio README.
- Real UI screenshot captured from mock/demo mode.
- Architecture and lifecycle diagrams.
- Demo notes.
- Public release guidance.

## Excluded

- Backend source code.
- Frontend source code.
- Database migrations.
- Docker configuration.
- Environment files.
- Secrets, tokens, credentials, or API keys.
- Internal-only notes.
- Customer or personal data.
- Local machine paths.
- Backup files.

## Recommendation

Use this repository for recruiter and hiring manager review. Keep the full `iran-car-yar` source repository private until a separate owner-approved source release is required.
